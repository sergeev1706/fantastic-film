Проект "Фантастика"

Проект представляет собой сайт, с описаниями фильмов жанра "Фантастика". Фильмы добавляются с помощью парсера.

Главная страница

На главной странице должны быть выведены 10 фильмов и пагинация. Для каждого фильма должна быть указана следующая информация: название, картинка, год выпуска, первые N слов описания, рейтинг. Также должна стоять ссылка на страницу с фильмом.

Страница фильма

На странице фильма должна быть указана следующая информация: название, картинка, год выпуска, полное описание, рейтинг, имя режиссера, актерский состав, ссылка на сторонний сайт для просмотра данного фильма.

Комментарии

На странице фильма пользователи могут оставлять комментарии к фильму, а также выставлять ему оценки, из которых затем формируется рейтинг фильма.

Фильтрация

Сайт должен позволять выполнять фильтрацию фильмов по следующим критериям: год выпуска, страна, рейтинг.

Наполнение

Необходимо написать парсер, который спарсит данные для сайта со сторонних источников.
